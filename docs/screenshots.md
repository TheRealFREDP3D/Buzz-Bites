## How to Add Screenshots

1. Take screenshots while playing the game
2. Place them in the `public/` directory
3. Update the paths in this file
4. Add to README.md sections

### Recommended Screenshot Tools
- **Windows**: Win + Shift + S (Snipping Tool)
- **Mac**: Cmd + Shift + 4 (Screenshot)
- **Browser**: Use developer tools for device emulation
